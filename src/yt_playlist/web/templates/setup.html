Pair the browser extension

Why an extension?

YouTube Music has no public, official API for managing your library and playlists. Google does offer an OAuth path (the YouTube Data API), and it can create and edit playlists, but it can't read your listening history, the "listens" data our recommendation models depend on, so it can't fully drive this app. So instead the app routes requests through a small browser extension (TuneConsole) running in your own signed-in browser session: the extension applies your cookies to each request and sends the response back over a local connection. This is the same underlying approach established tools like ytmusicapi take, out of necessity, just brokered live through the extension instead of a one-time pasted capture.

Your session cookies never leave your browser or cross this connection: the app sends only the request (method, URL, body), and the extension adds auth and sends the response back. Nothing credential-related is stored on disk here.

What does this give access to?

Only your YouTube Music library, playlists, and listening history. The extension uses the sign-in your browser already has for YouTube Music, but your session cookie never leaves the browser: it is applied to each request inside the page, and only the response comes back to this app. The app never receives or stores your cookie or any Google credential; it sends only the request (method, URL, body).

The connection is scoped to YouTube Music: the extension only signs requests to YouTube Music's own API and refuses anything else, so it cannot be used to reach Gmail, Drive, or other Google services. You can watch every request the app makes on the Network page (full details in the privacy policy). To revoke access, remove the extension or sign out of YouTube Music in the browser.

Install the TuneConsole extension and it connects automatically, there is nothing to paste. Keep a music.youtube.com tab open while signed in. The extension is recognized by its identity, so no other program can use this connection.
